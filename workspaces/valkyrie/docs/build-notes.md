# Build Notes — Project Valkyrie (CB750 Cafe Racer)

## Carburetors
Running the stock Keihin CV carbs, rejetted for the new pod filters and 4-into-1
exhaust. Bench-synced before install; will vacuum-sync once running.

## Charging
Converted to a modern regulator/rectifier. The stock points ignition was
replaced with an electronic ignition module to stabilize timing.
