# Cam Timing Notes — 2JZ-GTE

Set the engine to TDC on cylinder 1 before pulling the cam gears. The crank
pulley notch should line up with the 0 mark on the timing cover.

Intake and exhaust cam dots align with the marks on the front cap when timing is
correct. Double-check after rotating the engine two full turns by hand.

Belt tension: install the tensioner with the pin in, rotate the crank two turns,
then pull the pin. Do not pry on the belt to set tension.
