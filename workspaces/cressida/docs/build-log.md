# Build Log — Project Cressida

## Bottom end
Block was bored 0.5mm over and decked. New ARP main studs torqued to spec.
Plastigage on mains read within tolerance.

## Head
MLS head gasket installed dry. Head bolts (studs) torqued cold, then angle-torqued.
Valve lash checked on a cold engine.

## Boost
Running a single GT3582 at a target of 18 psi to start, will turn it up once the
tune is dialed in on the dyno.
